Bienvenido a la Herramienta de Detección de Fake News Multimodal

Esta herramienta utiliza un modelo avanzado de Machine Learning para identificar noticias falsas (fake news) utilizando tanto texto como imágenes. Al analizar conjuntamente el contenido textual y visual, nuestro modelo puede predecir con mayor precisión si una noticia es falsa o no. Simplemente ingresa un texto y carga una imagen relacionada, y nuestro sistema utilizará técnicas de aprendizaje profundo para evaluar la veracidad de la información presentada.
